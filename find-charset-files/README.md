# find-charset-files
Search file content by specified character set over folder tree.
Tested character sets:
* GB2312
* GBK
* UTF-8
* UTF-16BE
* UTF-16LE

# Usage
Windows:
```dos
find-charset-files.bat
```

Linux:
```bash
./find-charset-files.sh
```
